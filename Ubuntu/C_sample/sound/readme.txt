 基本計算

 fs = 1 / ts

 fs = サンプリング周波数
 ts = サンプリング周期


 [量子化]
 アナログ信号を数として記録すること。
量子化精度の単位をbitで表し、量子化精度を「Q」、ステップ数を「N」とすると、

 --	N = 2^Q
