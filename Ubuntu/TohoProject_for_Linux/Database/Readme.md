# 【ディレクトリ構成】
* **src**</br>
ソースコードを管理

* **include**</br>
各種データベースが記述されているヘッダーファイル

* **raw**</br>
プログラム実行時に出力されるrawファイルの保存先ディレクトリ。</br>
rawファイルはサイズがでかいためgitでは管理しない。
</br>
