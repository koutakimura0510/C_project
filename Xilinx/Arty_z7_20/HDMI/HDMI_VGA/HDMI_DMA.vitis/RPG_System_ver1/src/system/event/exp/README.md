# 【経験値テーブル】
## 【TOP】
レベルアップに必要な経験値ととレベルアップ時の習得技に関するデータをまとめたフォルダです</br>
</br>

## 【技の詳細】
各キャラ最大10この技を習得する。</br>
技の使用回数に応じて技の性能を上げていくことができる。</br>
